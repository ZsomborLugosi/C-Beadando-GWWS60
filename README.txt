Futtatáshoz szükséges technológiák:
    -Visual Studio/Jetbrains Rider/vscode + c#Devkit
    -.NET 8 SDK
    
Futtatási instrukció:
    A zip filet behúzzuk/megnyitjuk a fejlesztői környezetben, és a .snl file belépési pont segítségével futtatjuk
    Az SQLite adatbázis inicializálja és seedeli magát azzal nincs további teendő
