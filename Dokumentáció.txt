# C-Beadando-GWWS60

Kikötőbeli adatkezelő alkalmazás.
Segít az információk tárolásában és vizualizálásában

Követelmények: .NET 8 SDK
VS/Rider/Vscode(c#dev kit)

Technológiák:
backend: ASP.NET core
frontend: HTML + CSS
db: SQLite

----------------------------------

Szerkezeti felépítés: 

wwwroot:
    css: 
        -tartalmazza a pagere vonatkozo css -t
    js:
        -alapvetően generált javascript mappa üres
    lib:
        -alapvető bootstrap és jquery fileokat tartalmaz
        
Controllers:
    HomeController.cs:
        -tartalmazza az alap weboldalszintű funkciókat egy kontrollerosztályban
        -mint pl a logolás az error result és az Index
        
    ShipController.cs:
        -tartalmazza a hajókra vonatkozó Controllereket mint a Create, Edit, Delete
        -aszinkron függvényeket alkalmaz, és post metódusokat managel
        -itt található a pdf nyomtatásért felelős controller is a Manifest nevű taskban
 
Data:
    DbInitializer.cs:
        -ez a class felel az adatbázis inicializálásáért amikor először fut a program
        -ha üres az adatbázis (első indításkor valószínűleg az) akkor seedeli 4 teszt értékkel
    
    DockyardContext.cs:       
        -Az SQLite adatbázis megfelelő működéséhez szükséges context osztály getterekkel és setterekkel is ellátva
 
Migrations:
    InitialCreate.cs:
        -Ez a migrációs osztály létrehozza az adatbázis első tábláját, a Ships táblát
        - Az osztályt az Entity Framework Core automatikusan generálja
        
    DockyardContextModelSnapshot.cs:
        -Ez a snapshot osztály a jelenlegi adatmodell állapotát mutatja 
        -az Entity Framework automatikusan frissíti minden migráció után
       
Models:
    ErrorViewModel.cs:
        -Alapvetően generált osztály ami a hibakiírást működteti
        
    Ship.cs:
        -A hajó osztály mezői és getter setterei 
        -kiegészítve a hozzáadási formok és nyomtatás igényeinek megfelelően 
       
Views:
    Home:
        Index.cshtml:
            -A főoldal html elemeit tartalmazza gombokkal és cardokkal stb
        
        Privacy.cshtml:
            -Alapvetően generált fájl ami a privacy policy oldal html elemeit tartalmazza
   
    Shared:
        _Layout.cshtml:
            -A 2 oldal olyan html elemeit tartalmazza amik közösek
            -mint pl a header és footer
            
        _ValidationScriptsPartial.cshtml
            -jquery validation funkciókoz szükésges
    
        Error.cshtml
            -hibák mutatásához tartalmaz html elemeket
        
    Ship:
        Create.cshtml:
            -Hajók hozzáadásánál/készítésénél kellő html elemek
            -aszinkron rendereléshez szükséges javascript kód
        
        Delete.cshtml
            -Törlés html elemei és törlés post metódusa
           
        Edit.cshtml
            -Hajók szerkesztésére kellő html elemek
            -és aszinkron rendering javascript
            
        Index.cshtml
            -Hajókezelő oldal html elemeit tartalmazza
            -sorok és gombok animációját végző javascript
           
    _ViewImports és _ViewStart .cshtml:
        - alapvető fileok amiket a projekt készítésekor generál 

.gitignore
    -biztosítja hogy csak azok a mappák és fájlok kerüljenek fel a github repoba amik oda valóak
  
appsettings.json
    -az adatbázishoz való kapcsolódáshoz szükséges JSON fileok

global.json
    -biztosítja a .NET verzió egyezést és hogy milyen verziók megengedettek
    
Program.cs:
    -A program belépőpontja és buildere van ebben a fileban
    -serviceket biztosítja hogy hozzá vannak adva
    -Adatbázis kapcsolódást biztosít
    -HTTP pipeline konfiguráció
    -A migration használatát biztosítja
    -routingokat intézi
    -és futtatja a programot
